Settings

Prowlarr

Connect to Prowlarr for torrent search via your indexers. Find your API key in Prowlarr > Settings > General.

qBittorrent

Connect to qBittorrent for torrent and audiobook downloads. Enable the Web UI in qBittorrent settings.

Audiobookshelf

Connect to Audiobookshelf for library browsing and audiobook management. Get your API token from Audiobookshelf > Settings > Users > your user.

Calibre-Web

Auto-import downloaded ebooks into Calibre-Web via calibredb. Requires Docker socket access.

lightnovel-crawler

Scrape web novels chapter-by-chapter into EPUB format. Requires the lncrawl Docker container.

Kavita

Import ebooks into Kavita by triggering a library scan. Requires OPDS API key.

Komga

Connect to Komga for manga and comic library management. Komga receives CBZ files after download.

Import Targets

Choose which library apps receive imported books. Enabled targets are called after every download.

Optional. Filters enabled targets by media type and/or source (e.g. {\"source\":{\"annas\":[\"calibre\"]}}).

File Organization

Organize downloaded files into Author/Title/ folder structure.

Download Paths

Where downloaded files are stored before organization.

Authentication

Protect Librarr with a username and password. Leave username blank to disable authentication. External tools can use the API key with the X-Api-Key header.

Import from Goodreads / StoryGraph

Export your "To Read" shelf from Goodreads (My Books → Export to CSV) or StoryGraph (Settings → Export Data), then upload the CSV here to batch-queue downloads.
